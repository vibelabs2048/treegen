How TreeGen Works

1. Select a box in the chart or use the default root person.

2. Edit person details in the inspector on the right.

3. Use Styles to adjust defaults for a generation.

4. Import or export YAML to save and continue later.

5. Use the desktop app when you need exact PDF, PNG, or JPG output.
