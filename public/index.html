How TreeGen Works

1. Use New Project for a blank tree or Load Demo for the sample family.

2. Use Open Project to load a saved TreeGen YAML project file.

3. Click any box in the chart, then edit person details in the inspector on the right.

4. Use Format and Title to adjust generation defaults and title-box content.

5. Use Save, Save As, and Autosave to manage your project while you work.

6. In the browser, Save As names the project file and Recent Projects lets you reopen or rename saved browser-side entries.

7. Use Recent Projects to reopen desktop project files quickly too.

8. Use the desktop app when you need exact PDF, PNG, or JPG output.
